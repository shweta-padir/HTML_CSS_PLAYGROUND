how to make hover effect on button
ChatGPT said:

To make a hover effect on a button, you use the :hover pseudo-class in CSS.

Here’s a simple example using your button styles:

css
button {
  height: 30px;
  width: 100px;
  color: white;
  background-color: blue;
  border: none;
  border-radius: 5px;
  box-shadow: 2px 2px 10px blue;
  cursor: pointer;
  transition: all 0.3s ease; /* Smooth animation */
}

button:hover {
  background-color: darkblue;  /* Changes color on hover */
  transform: scale(1.05);      /* Slightly grows button */
  box-shadow: 2px 2px 15px darkblue; /* Shadow changes */
}

What’s happening:

  • transition: all 0.3s ease; → makes the change smooth.

  • background-color: darkblue; → gives a darker color on hover.

  • transform: scale(1.05); → slightly enlarges the button when hovered.

  • box-shadow changes for a glowing effect.
